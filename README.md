# AudioMixer + Kick Drum Circuits
I've been interested in analog electronics and electronic instruments for a decent chunk of my time in college so I wanted to try my hand at creating an [audio mixer](https://youtu.be/q8tmUgaXrEQ?si=pKyhKzkajhBQ0o0y&t=1063) and a [kick drum circuit](https://youtu.be/yz37Yz315eU?si=P70lOI3g9exFc0Ud&t=1760) using designs I found online. This will be the first part of a bigger project where I hope to make a few different analog synthesizer modules and use the mixer to, well, mix them together. :)

### Audio Mixer Design
![alt text](https://github.com/Jumpin-Josh/AudioMixer/blob/main/simulations/Mixer%20Prototype.png "Initial Mixer Design in LTSpice")
Each of the 100k potentiometers is setup in a voltage divider configuration allowing each input to be attenuated. The inputs are then tied together and fed into two inverting amplifiers to boost the resulting signal since the input pots may reduce the amplitude. The 20k potentiometer is used to add/adjust clipping on the mixer's output which is then fed into another op-amp since the clipping reduces the signal amplitude yet again. (Each potentiometer can be swept through using a .step directive to see how different values will affect the waveform.)
